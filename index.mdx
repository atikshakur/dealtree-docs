# Dealtree user guide

Use this documentation to move from initial setup to account execution with a repeatable process.

<Steps>
  <Step title="Set your seller profile">
    Define product name, category, and description so Dealtree can tailor stakeholder and action recommendations.
  </Step>
  <Step title="Create and prepare an account">
    Add the target account, then capture known context in notes.
  </Step>
  <Step title="Generate account intelligence">
    Sync org data, review stakeholders, and generate buying committee roles.
  </Step>
  <Step title="Execute the action plan">
    Prioritize recommended tasks, update notes after interactions, and regenerate when context changes.
  </Step>
</Steps>
